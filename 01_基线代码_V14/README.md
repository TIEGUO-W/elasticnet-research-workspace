# 01 — 基线代码（V14）

## 文件说明

| 文件 | 说明 |
|------|------|
| `V14绘图优化版.py` | 锅哥原始的弹性网络反演辐射源活度代码（鲁棒优化版 v5.1-Clean） |

## 这段代码做了什么

1. 用 PHITS 模拟数据（或合成数据）构建 8×12 系统矩阵 A
2. 用 sklearn 的 ElasticNet + GridSearchCV 做超参数搜索
3. 自定义 scorer 带截距惩罚项
4. 导出结果到 CSV/Excel + 画对比图

## 核心参数

- 矩阵：m=8（探测器）× n=10（源位）
- 稀疏度：0.6（60%的源为零）
- 搜索范围：α ∈ [1e-6, 1e-1], l1_ratio ∈ [1e-10, 1-1e-10]
- CV 折数：5

## 局限性

- 没有对比基线（OMP/Lasso/Ridge）
- 无不确定性量化
- 截距惩罚阈值是硬编码魔数
