# 03 — 方案B：混合源分布重建（双轨验证）

## 要回答的问题

> 在真实 PHITS 数据上（8×12矩阵，cond=3.1e21），各种方法能恢复几个源？放在"不可观测区域"的源会怎样？

## 双轨设计

- **Track 1（合成数据）**：8×10 矩阵，可控条件数（10~1000），验证方法本身
- **Track 2（PHITS真实数据）**：8×12 矩阵，5个零列故意放非零值，测试"不可观测"场景

## 核心发现

**Track 1（合成数据）**：Cond=300 时 Lasso/EN/OMP 都能 100% 支持恢复

**Track 2（PHITS）**：
- 零列放非零值时，所有方法都**无法感知**这些源
- 有噪声时 MSE 飙升到 1e4~1e7（cond=3.1e21 太大）
- 结论：PHITS 8×12 数据不适合做核心验证
