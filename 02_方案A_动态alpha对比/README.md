# 02 — 方案A：动态α vs 固定α 对比实验

## 要回答的问题

> Elastic Net 迭代过程中动态调整 L1/L2 权重（早期L2主导，后期L1主导），是否比固定权重更好？

## 结论：❌ 动态α被否决

动态α（logistic调度）在所有条件数下 MSE 比固定α差 1-3 个数量级，支持恢复率不到 50%。

| 文件 | 说明 |

## 核心数据（Cond=300，50次平均）

| 方法 | MSE | 支持恢复率 | 相对误差 |
|------|-----|-----------|---------|
| **EN_Fixed（固定α）** | **0.042** | **100%** | **0.065** |
| EN_Dynamic（动态α） | 0.663 | 47.5% | 0.962 |
| Lasso | 0.014 | 100% | 0.029 |
| OMP | 0.358 | 85% | 0.300 |
| Ridge | 0.193 | 100% | 0.505 |

## 注意

- 这是**合成数据**结果（8×10矩阵，可控条件数），不是 PHITS 真实数据
- Lasso 在合成数据上表现最好，但 PHITS 场景（极弱列）不一定如此
- 跑了3轮（3个时间戳），只保留了最新一轮
